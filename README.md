
## Coffinlink

A script for more detailed testing of links.



![Logo](https://i.ibb.co/F6GX0cv/free-icon-link-3876194.png)


## Run Locally

Clone the project

```bash
  git clone https://github.com/Kaiserflexer/coffinlink.git
```

Go to the project directory

```bash
  cd coffinlink
```

Start the script 

```bash
  go run main.go
```
